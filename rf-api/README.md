## Ruffled Feathers
### A bird app for people who love birds.
---

### User Story
As a user, I want to...
* create an account.
* create a bird sighting log
* filter sightings by location or bird name
* favorite birds and sightings
* leave comments on others' bird sightings
* add images to my sightings
